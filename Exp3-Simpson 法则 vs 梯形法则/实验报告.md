2. 通过实验比较两种方法在积分精度和误差方面的表现。
3. 能够分析不同子区间数 $N$ 对积分结果的影响。

## 二、实验原理

### 2.1 Simpson 法则

简述 Simpson 法则的积分思想、公式推导及适用条件（$N$ 必须为偶数），并给出公式：
Simpson 法则的积分思想是用二次函数来近似被积函数。在每个子区间 \([x_{i}, x_{i + 2}]\) 上，构造一个通过三个点 \((x_{i}, f(x_{i}))\)、\((x_{i + 1}, f(x_{i + 1}))\)、\((x_{i + 2}, f(x_{i + 2}))\) 的二次多项式 \(p(x)\) ，然后对该二次多项式在子区间上积分来近似原函数的积分。公式推导：设二次多项式 \(p(x) = Ax^2 + Bx + C\) ，在区间 \([x_{i}, x_{i + 2}]\) 上积分，经过一系列计算可得子区间积分近似值。将所有子区间积分相加，得到整个区间 \([a, b]\) 上的积分近似公式。适用条件是子区间数 N 必须为偶数 。公式为：
\(\int_a^b f(x)dx \approx \frac{h}{3}\left[f(x_0) + 4\sum_{i = 1,3,5,\cdots}^{N - 1}f(x_i) + 2\sum_{i = 2,4,6,\cdots}^{N - 2}f(x_i) + f(x_N)\right]\)
其中 \(h = \frac{b - a}{N}\) ，\(x_i = a + ih\) 。
$$
\int_a^b f(x)dx \approx \frac{h}{3}\left[f(x_0) + 4\sum_{i=1,3,5,...}^{N-1}f(x_i) + 2\sum_{i=2,4,6,...}^{N-2}f(x_i) + f(x_N)\right]
$$

### 2.2 梯形法则

简述梯形法则的积分思想、公式推导，并给出公式：
梯形法则的积分思想是将积分区间 \([a, b]\) 划分为 N 个子区间，在每个子区间上用梯形的面积来近似小曲边梯形的面积。公式推导：对于第 i 个子区间 \([x_{i}, x_{i + 1}]\) ，梯形面积为 \(\frac{h}{2}[f(x_{i}) + f(x_{i + 1})]\) ，将所有子区间梯形面积相加，得到积分近似公式：
\(\int_a^b f(x)dx \approx h\left[\frac{f(x_0)}{2} + f(x_1) + \cdots + f(x_{N - 1}) + \frac{f(x_N)}{2}\right]\)
其中 \(h = \frac{b - a}{N}\) ，\(x_i = a + ih\) 。
$$
\int_a^b f(x)dx \approx h\left[\frac{f(x_0)}{2} + f(x_1) + \cdots + f(x_{N-1}) + \frac{f(x_N)}{2}\right]
$$

## 三、实验代码

请粘贴你的完整 Python 实现代码（包括 Simpson 法则和梯形法则）：

```python
import numpy as np

# 待积分函数
def f(x):
    return x**4 - 2*x + 1

# 已给出的梯形法则积分函数，供参考比较用
def trapezoidal(f, a, b, N):
    h = (b - a) / N
    x = np.linspace(a, b, N + 1)
    fx = f(x)
    integral = h * (0.5 * fx[0] + np.sum(fx[1:-1]) + 0.5 * fx[-1])
    return integral

# Simpson法则积分函数
def simpson(f, a, b, N):
    if N % 2 != 0:
        raise ValueError("Simpson法则要求N必须为偶数")
    h = (b - a) / N
    x = np.linspace(a, b, N + 1)
    fx = f(x)
    # 奇数索引（1,3,5,...,N - 1）
    odd_sum = np.sum(fx[1:N:2])
    # 偶数索引（2,4,6,...,N - 2）
    even_sum = np.sum(fx[2:N:2])
    integral = h / 3 * (fx[0] + 4 * odd_sum + 2 * even_sum + fx[N])
    return integral

def main():
    a, b = 0, 2
    exact_integral = 4.4

    for N in [100, 1000]:
        trapezoidal_result = trapezoidal(f, a, b, N)
        simpson_result = simpson(f, a, b, N)

        trapezoidal_error = abs(trapezoidal_result - exact_integral) / exact_integral
        simpson_error = abs(simpson_result - exact_integral) / exact_integral

        print(f"N = {N}")
        print(f"梯形法则结果: {trapezoidal_result:.8f}, 相对误差: {trapezoidal_error:.2e}")
        print(f"Simpson法则结果: {simpson_result:.8f}, 相对误差: {simpson_error:.2e}")
        print("-" * 40)

if __name__ == '__main__':
    main()
```

### 4.2 误差分析
- 分析两种方法在不同 $N$ 下的误差表现。
- 不同 N 下的误差表现：当 \(N = 100\) 时，梯形法则相对误差为 \(1.22e - 03\) ，Simpson 法则相对误差为 \(0.00e + 00\) ；当 \(N = 1000\) 时，梯形法则相对误差减小到 \(1.22e - 05\) ，Simpson 法则相对误差仍为 \(0.00e + 00\) 。
- Simpson 法则与梯形法则误差的数量级差异及原因。
- Simpson 法则误差数量级通常比梯形法则小。这是因为 Simpson 法则用二次函数近似被积函数，能更好拟合曲线，而梯形法则只是用直线（梯形）近似，精度相对较低。
- 讨论 $N$ 增大时误差的变化趋势。
N 增大时误差的变化趋势：随着 N 增大，两种方法误差都趋近于 0 。梯形法则误差呈幂次下降趋势，Simpson 法则在合适条件下（N 为偶数且足够大）可快速得到精确结果。
## 五、结论
- 总结两种方法的适用场景和优劣。
- 梯形法则：优点是原理简单、实现容易，适用于对精度要求不高、被积函数变化较平缓的情况。缺点是精度相对较低，对于复杂函数或要求高精度积分时，需大量子区间才能达到较好效果。Simpson 法则：优点是精度高，尤其对于光滑函数能快速得到高精度结果。缺点是要求子区间数 N 为偶数，适用条件相对受限，若函数不光滑，优势可能不明显。
- 结合本实验结果，说明实际工程中如何选择数值积分方法。
- 在实际工程中，若对精度要求不高、计算资源有限且被积函数变化平缓，可选择梯形法则，因其简单易实现。若对精度要求较高、被积函数较为光滑，应优先选择 Simpson 法则，可在较少计算量下获得高精度结果。同时，还需综合考虑计算效率、函数特性等因素，必要时可结合多种数值积分方法使用。
